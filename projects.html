Projects

Here I am showcasing three of my playable independently developed video games, websites I have built and deployed, and mobile applications built for iPhone. Links are provided at the bottom for all my projects, including the ones not showcased here.

Games

The games below were coded in C# through Visual Studio and I developed them in Unity. Every asset was made from scratch, and the artwork was created using Adobe Illustrator. If you're mildly interested, I have other games available on my itch.io page, including platformers, word games, and horror titles.

Wake Him Up
Casual physics-based puzzle: Manipulate the wind to guide a falling feather through the bedroom window and wake up the procrastinator by dodging balls, kites, balloons, and more! The game was made to match the theme "Just Get Started".

Roundabout
Casual puzzle: Control a car that drives around silly objects, like donuts and bagels, and dodge whatever blocks your path by adjusting the speed mode with precise timing. This one is easy to start, but quite tricky to master. Also, it's mildly addictive.

Lost in Aurantius
Top-down adventure game: You find yourself on an alien planet and need to settle into the community. There are various tasks like chatting, fishing, and fruit picking. My teammate and I developed this game for Mental Health Dev Champions 2024 jam.

Web Development

Below are websites that have I built and deployed. They are hosted live on GitHub pages. The projects were built in VS Code and I have used JavaScript, HTML, CSS, and React (Vite). The code is available on my GitHub.

Quote Generator

Portfolio
HTML, CSS: This portfolio, showcasing my personal projects, was made with HTML and for styling I used CSS.

Mobile Development

Below are mobile applications built for iOS (specifically the iPhone). The code, assets, and screenshots are available on my GitHub.

To-do list
Swift: This is an iOS application that provides a to-do list with capability to add, edit, delete and prioritise tasks. The tasks are saved and loaded from a JSON file. I made use of Git version control.

Sketch It Out
Swift: This is an iOS application for casual sketching. Users are given random objects to draw and have an option to save them for later use. Here, I made use of Swift Databases and Git version control.
